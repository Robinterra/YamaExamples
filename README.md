Some examples in Yama.

## Json in Yama
### Raspberry Pi
Go in the folder with JsonYama
```
cd JsonYama
```
Build .yexe with YamaCompiler
```
YamaCompiler build
```
Run out.yexe
```
chmod u+x bin/out.yexe
./bin/out.yexe
```
### Every other System
Go in the folder with JsonYama
```
cd JsonYama
```
Build .yexe with YamaCompiler
```
YamaCompiler build
```
Run out.yexe
```
YamaCompiler ./bin/out.yexe
```
